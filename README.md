### Requirements

- Python 3.12

### Setup

```powershell
cd <path>
```

1. Install required packages:
```powershell
python -m pip install -r requirements.txt
```

2. Copy `.env.example` to `.env`, and add your MySQL credentials.

3. Run setup script:
```powershell
python setup.py
```
