\doxysubsection{Detailed Description}
Lightweight container for a span of story text. 


\begin{DoxyItemize}
\item Carries positional metadata so downstream consumers can reconstruct context.
\item Filter by story\+\_\+id to fetch all chunks for a particular story.
\item Use story\+\_\+percent and chapter\+\_\+percent to quickly sort chunks by intended order.
\item Use book\+\_\+id, chapter\+\_\+number, line\+\_\+start, and line\+\_\+end to locate this chunk within source material. 
\end{DoxyItemize}
