\doxysubsection{Detailed Description}
Stores active database connections and configuration settings. 


\begin{DoxyItemize}
\item This class implements Singleton design, so only one session can be created.
\item However, the session config can still be updated using the normal constructor. 
\end{DoxyItemize}
