Splits the target book into paragraphs. 

Yields Chunk objects for each paragraph (

) in the TEI file. Uses etree Element.\+sourceline to approximate start/end line in TEI. Supports optional start\+\_\+inclusive / end\+\_\+inclusive boundaries to slice text and stop iteration. Computes progress percentages using character counts\+:
