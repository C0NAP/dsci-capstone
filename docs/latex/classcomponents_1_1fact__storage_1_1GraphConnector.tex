\doxysubsection{Detailed Description}
Connector for Neo4j (graph database). 


\begin{DoxyItemize}
\item Uses neomodel to abstract some operations, but raw CQL is required for many tasks.
\item Neo4j does not support multiple logical databases in community edition, so we emulate them using a {\ttfamily database\+\_\+id} property on nodes. 
\end{DoxyItemize}
